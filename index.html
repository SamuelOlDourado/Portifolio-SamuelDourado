Desenvolvedor Web | HTML5, CSS3, Bootstrap, Python

Sou desenvolvedor web com foco na criação de interfaces responsivas, acessíveis e visualmente bem estruturadas. Trabalho principalmente com HTML5, CSS3, Bootstrap 5 e JavaScript, sempre buscando unir clareza visual e organização de código.

Tenho experiência no desenvolvimento de aplicações web completas, atuando tanto no front-end quanto no back-end com Flask, integrando bancos de dados e implementando funcionalidades como autenticação e operações CRUD.

Antes de iniciar o desenvolvimento, costumo planejar os layouts no Figma, garantindo uma boa experiência do usuário desde a estrutura até os detalhes finais. Meu objetivo é criar soluções web eficientes, acessíveis e fáceis de manter.

Sobre Mim

Minha Jornada

Meu interesse por tecnologia surgiu cedo, motivado pela curiosidade em entender como as coisas funcionam por trás das interfaces. Em 2024, iniciei oficialmente minha trajetória no desenvolvimento de sistemas, entrando de vez no universo do desenvolvimento web e do web design.

Desde então, venho evoluindo de forma contínua, aplicando o aprendizado em projetos práticos e aprimorando tanto a lógica de desenvolvimento quanto a construção de interfaces organizadas e funcionais.

Atualmente, estou focado em aprofundar meus conhecimentos em desenvolvimento web e web design, explorando JavaScript, CSS moderno, organização de layouts e boas práticas voltadas à experiência do usuário.

Metas e Objetivos

Busco oportunidades para aplicar e expandir meus conhecimentos em desenvolvimento web front-end, contribuindo com projetos desafiadores que impactem positivamente os usuários.
